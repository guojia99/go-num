## 描述

等差数列 2，5，8，11，14。。。。

（从 2 开始的 3 为公差的等差数列）

输出求等差数列前n项和

本题有多组输入

数据范围： ![img](https://www.nowcoder.com/equation?tex=1%20%5Cle%20%20n%20%5Cle%201000%20%5C)

### 输入描述：

输入一个正整数n。

### 输出描述：

输出一个相加后的整数。

## 示例1

输入：

```bash
2
```

复制

输出：

```bash
7
```

复制

说明：

```bash
2+5=7 
```

## 示例2

输入：

```bash
275
```

复制

输出：

```bash
113575
```

复制

说明：

```bash
2+5+...+821+824=113575
```

## 思路

观察规律，或者想想是否还记得公式。

## 代码

```js
while(n = ~~readline()) {
  console.log(2 * n + (n-1)*n/2*3)
}
```
