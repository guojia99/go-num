## 描述

输入一个整数，将这个整数以字符串的形式逆序输出

程序不考虑负数的情况，若数字含有0，则逆序形式也含有0，如输入为100，则输出为001

数据范围： ![img](https://www.nowcoder.com/equation?tex=0%20%5Cle%20n%20%5Cle%20%202%5E%7B30%7D-1%20%5C)

### 输入描述：

输入一个int整数

### 输出描述：

将这个整数以字符串的形式逆序输出

## 示例1

输入：

```bash
1516000
```



输出：

```bash
0006151
```



## 示例2

输入：

```bash
0
```



输出：

```bash
0
```

## 思路

- 基础字符串数组互转翻转处理


## 代码

```js
while(num = readline()) {
  console.log([...num].reverse().join(''))
}
```
